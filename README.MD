## 🚀 Основні Фічі
- **📚 Симуляція:** Проходження реальних тестів минулих років (з таймером).
- **🧠 AI Пояснення:** Генерація детальних рішень через **Gemini 3 Flash Pro**.
- **📊 Статистика:** Відстеження прогресу та рейтингу (НМТ-шкала 100-200).
- **� Адмінка:** Керування питаннями, користувачами та розсилками.
- **� Розумний UI:** Автоматичне приховання фото для довгих пояснень >1024 символів.

## 🛠 Технології
- **Python 3.11** + `aiogram 3.x` + `aiogram-dialog`
- **PostgreSQL** + `SQLAlchemy` + `Alembic`
- **Redis** (FSM Storage)
- **Docker Compose**

## ⚙️ Установка та Запуск

### 1. Налаштування
Створіть `.env` файл (див. `.env.dist`):
```bash
cp .env.dist .env
# Заповніть BOT_TOKEN, GEMINI_API_KEY і т.д.
```

### 2. Запуск (Docker)
```bash
# Локально або на сервері
docker compose up -d --build
```

### 3. Деплой
Для деплою на віддалений сервер (Hetzner/DigitalOcean):
```bash
./scripts/remote_deploy.sh
```
