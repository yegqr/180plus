# 🛠️ Services Layer

Цей пакет містить допоміжні сервіси та класи, які реалізують конкретні технічні завдання, винесені за межі хендлерів.

## 📁 Основні сервіси

*   [`broadcaster.py`](file:///Users/yegqr/NMT-bot/tgbot/services/broadcaster.py) — **Сервіс розсилки повідомлень**.
    *   Реалізує логіку безпечної розсилки великій кількості користувачів.
    *   Підтримує обробку помилок (блокування боту, видалення аккаунту).
    *   Використовується як при старті боту (сповіщення адмінів), так і в діалозі розсилок.

## 🚀 Використання

Сервіси зазвичай є набором асинхронних функцій або класів, які приймають об'єкт `Bot` та необхідні дані.
Приклад виклику розсилки:
```python
from tgbot.services.broadcaster import broadcast
await broadcast(bot, users_ids, "Текст повідомлення")
```
